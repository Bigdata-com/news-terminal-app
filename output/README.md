# Report Outputs

This directory contains AI-generated reports from the `cli_report_generator.py` script.

## File Types

### 1. Executive Briefs (`*_briefs.txt`)
Concise bullet points covering key developments by topic. Perfect for quick executive review.

**Format:**
- One bullet point per topic
- Bold company name
- Specific metrics and dates
- 25-40 words per brief

### 2. Wall Street Desk Note (`*_desk_note.txt`)
Cohesive narrative synthesizing all briefs into a professional desk note format.

**Format:**
- Wall Street analyst style
- Multiple focused paragraphs
- Dense, information-rich content
- Industry-standard abbreviations

### 3. Full Report JSON (`*_full_report.json`)
Complete structured data including all briefs, desk note, and metadata.

**Contains:**
- All briefs with topics
- Complete desk note
- Search statistics
- Article counts
- Generation timestamp

## File Naming Convention

```
{TICKER}_{YYYYMMDD_HHMMSS}_{type}.{ext}

Examples:
  TSLA_20251029_143022_briefs.txt
  TSLA_20251029_143022_desk_note.txt
  TSLA_20251029_143022_full_report.json
```

## Usage

Generated by:
```bash
python scripts/cli_report_generator.py TICKER [options]
```

See `python scripts/cli_report_generator.py --help` for full options.
